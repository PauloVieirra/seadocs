
# SEADOCS - Sistema de Gerenciamento Inteligente de Documentos

Sistema de gerenciamento de documentos com IA integrada, controle de permissões por níveis de sigilo e interface moderna.

## Funcionalidades

- **Criação e edição de documentos** com modelos personalizáveis
- **Controle de permissões** por níveis de sigilo (Público, Restrito, Confidencial)
- **Busca inteligente** de usuários e documentos
- **Integração com IA** para geração de conteúdo
- **Gestão de projetos e grupos**
- **Auditoria completa** de ações do sistema

## Instalação e Execução

### Pré-requisitos
- Node.js 18+
- npm ou yarn
- Supabase (para banco de dados)

### Instalação
```bash
npm install
```

### Desenvolvimento
```bash
npm run dev
```

### Build para produção
```bash
npm run build
```

## Estrutura do Projeto

- `src/app/components/` - Componentes React da interface
- `src/services/` - Serviços de API e integração
- `src/styles/` - Estilos e temas
- `src/main.tsx` - Ponto de entrada da aplicação

## Tecnologias Utilizadas

- **React 18** - Framework frontend
- **TypeScript** - Tipagem estática
- **Tailwind CSS** - Estilização
- **Supabase** - Backend e banco de dados
- **shadcn/ui** - Componentes de UI
- **Vite** - Build tool
